This application simulates a chaos game. The game starts with several static points (shown in pink here) and a single moving point (in black). In the first round, one of the static points is chosen at random. The moving point then moves halfway towards that point, leaving a dot behind. In each succeeding round, a static point which is not the same one as in the previous round is chosen at random, and the moving point again moves halfway towards it.
Controls:
  • UP ARROW: Add a static point
  • DOWN ARROW: Delete the static point the cursor is hovering over
  • LEFT CLICK: Start the moving point at the cursor's position
